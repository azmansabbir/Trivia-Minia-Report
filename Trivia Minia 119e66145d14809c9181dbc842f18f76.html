Trivia Minia

  • Trivia-001 | Steps to Create an Instance in Trivia-Minia Game
    • Precondition: The user is logged in as an admin.
    • Steps:
    1. Navigate to the Trivia-Minia game.
    1. Click on the “Instances” plus button.
    1. Input a valid instance name (e.g., "Trivia 1").
    1. Click the save button.

    Expected Result: The admin sees the newly created instance "Trivia 1" in the instances list.

  • Trivia-002 | Steps to Add New Round inside an Instance
    • Precondition:
      • The user is logged in as an admin.
      • There should be at least one existing instance.
    • Steps:
      1. Navigate to the instance “Control” panel.
      1. Click on the “Control” button.
      1. Click on the “Add Round” button.
      1. Input a valid round name (e.g., "Round 1").
      1. Click the Add button.
    • Expected Result: After clicking the Add button, the user should see a confirmation message indicating that the round has been successfully added. Additionally, the new round (e.g., "Round 1") should be displayed in the list of rounds within the instance “Control” panel.
  • Trivia-003 | Steps to Add Multiple Choice Questions Inside a Round

    Precondition

    • The user is logged in as an admin.
    • There should be at least one existing instance.
    • Inside the instance, there should be at least one round.

    Steps

    1. Navigate to the instance “Round” section.
    1. Click on the “Add Question” button.
    1. Click on the “Multiple Choice” radio button.
    1. Input a valid question name (e.g., "Question 1").
    1. Enable the “Disable Media Frame” button.
    1. Upload images for “Mobile” and “Mainboard Media.”
    1. Click to select the arrangement as “Auto.”
    1. Click on the “Add Answer” button.
    1. Input the answer title (e.g., "Answer 1").
    1. Upload the answer image.
    1. Click the save button.
    1. Click on the “Add Answer” button (Note: for multiple choice, two answers are needed for selection).
    1. Input the answer title (e.g., "Answer 2").
    1. Upload the answer image.
    1. Click the save button.
    1. Click to save the question.

    Expected Result

    • After completing these steps, the user should see a confirmation message indicating that the question has been successfully added. The newly created question (e.g., "Question 1") with its associated answers should be displayed in the instance’s “Round” section. This verification should be possible on both mobile gameplay and the mainboard.
  • Trivia-004 | Validate Multiple Choice Question Successfully Showing on Game Play And Mainboard

    Precondition

    • The user is logged in as an admin.
    • There should be at least one existing instance.
    • Inside the instance, there should be at least one round.

    Steps

    1. Navigate to the instance section.
    1. Click on the “Start” button.
    1. Click on the “Yes” button to confirm the game start process.
    1. Click to “Move to Next Stage” on “Question” stage
    1. Click on the “QR” icon
    1. Click on the “Open Link” button
